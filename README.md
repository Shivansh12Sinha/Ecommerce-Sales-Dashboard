# E-Commerce Sales Dashboard

## Project Overview
This project analyzes e-commerce sales data using Tableau to uncover business insights related to sales performance, profitability, customer behavior, and regional trends.

## Tools Used
- Excel
- Tableau Public

## Key Features
- KPI Cards for Sales, Profit, and Orders
- Sales by Category Analysis
- Profit by Region Visualization
- Monthly Sales Trend Analysis
- Top Customer Identification
- Interactive Dashboard Filters

## Business Insights
- Technology category generated the highest sales.
- Western region produced maximum profit.
- Sales trends indicated seasonal fluctuations.
- Top customers contributed heavily to revenue.
